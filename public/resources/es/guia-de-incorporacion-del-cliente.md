# Guia de Incorporacion del Cliente

1. Programe la conversacion inicial.
2. Confirme el alcance del servicio y flujo mensual.
3. Proporcione acceso al software de contabilidad.
4. Comparta de forma segura documentos bancarios, tarjetas, prestamos, nomina y proyectos.
5. Identifique proyectos activos y necesidades de reportes.
6. Revise el primer plan de limpieza o cierre mensual.
7. Confirme comunicacion y fechas limite.

Hay documentos en espanol y asistencia de idioma cuando este disponible. Steel Beam no se anuncia como una oficina completamente bilingue.
