# Manual del Cliente

Use este manual para mantener el trabajo mensual organizado.

- Envie estados de cuenta, recibos, registros de nomina y documentos de proyecto a tiempo.
- Marque transacciones inusuales antes del cierre mensual.
- Mantenga gastos personales y del negocio separados cuando sea posible.
- Avise a Steel Beam sobre nuevos prestamos, compras de equipo, proyectos importantes o cambios de software.
- Revise reportes cada mes y haga preguntas temprano.
- Use nombres de proyectos de forma consistente para mejores reportes.
